Forward Kinematics and Velocity Update

To simluate the film movement over time, we implement a forward kinematics step that integrates vertex positions from their velocities and updates the velocities from forces derived from a cotangent-weighted Laplacian operator. It require building a mass matrix \( M \), a Laplacian matrix \( L \), and a position matrix \( X \), then integrates forward in time using a semi-implicit scheme. Let \( n \) denote the number of vertices. Each vertex i has position \( \mathbf{p}_i \) and velocity \( \mathbf{v}_i \).

Mass Matrix

We compute a diagonal inverse mass matrix \( \mathbf{M}^{-1} \) using barycentric area distribution. For a triangular face with vertices i,j,k, the area contribution is

\[ \text{share} = \big\|(\mathbf{p}_j - \mathbf{p}_i) \times (\mathbf{p}_k - \mathbf{p}_i)\big\|. \]

Each vertex's mass is computed from the sum of one-third of the area of its incident triangles.The inverse mass matrix is computed as:


\( M_{ii}^{-1} = \frac{6}{\max(\text{mass}_i, \varepsilon)} \)

where \( \text{mass}_i \) is the sum of the areas of adjacent triangles. The factor of 6 accounts for barycentric weighting: each triangle contributes \( \frac{1}{3} \) of its area to each vertex, and the area computation does not account the usual \( \frac{1}{2} \) factor, hence resulting in an overestimation by a factor of 6.

Velocity and Position Update

We compute accelerations from the Laplacian acting on the position matrix \( \mathbf{X} \) and integrate velocities explicitly. Using timestep \( \Delta t \):

\[ \mathbf{a} = \mathbf{M}^{-1} \mathbf{L} \mathbf{X} \]
\[ \mathbf{V}(t + \Delta t) = \mathbf{V}(t) + \Delta t \, \mathbf{a} \]
\[ \mathbf{X}(t + \Delta t) = \mathbf{X}(t) + \Delta t \, \mathbf{V}(t + \Delta t) \]

Bubble BSDF

In rendering the bubbles for this project, we extended the HW3 code to incorporate physically accurate thin-film transmission and reflection. Our first step was to implement a helper function thin_film_RT to compute the intensities of light that are reflected and refracted by the bubble’s surface. We specified the film’s refractive index 𝑛𝑓, its thickness 𝑑, and the refractive index of air 𝑛0. Using these parameters, along with the light’s wavelength λ and incident angle 𝜃0, we first applied Snell’s law to calculate the film angle 𝜃𝑓.

\[ n_0 · sin θ_0 = n_f · sin θ_f \] \[ cos θ_f = √ [ 1 − (n_0/n_f)^2 · (1 − cos^2θ_0) ] \]

Thin film phase retardation:

\[δ = (2π · n_f · d · cos θ_f) / λ\]

We then applied the following formulas to compute the Fresnel reflection (r) and transmission (t) amplitude coefficients for both s polarization (vertical) and p-polarization (parallel).

\[ r_{01}^{s} = \frac{n_{0} \cos\theta_{0} - n_{f} \cos\theta_{f}} {n_{0} \cos\theta_{0} + n_{f} \cos\theta_{f}} \] \[ r_{01}^{p} = \frac{n_{f} \cos\theta_{0} - n_{0} \cos\theta_{f}} {n_{f} \cos\theta_{0} + n_{0} \cos\theta_{f}} \]
\[ t_{01}^{s} = \frac{ 2\,n_{0} \cos\theta_{0} } { n_{0} \cos\theta_{0} + n_{f} \cos\theta_{f} } \] \[ t_{01}^{p} = \frac{ 2\,n_{0} \cos\theta_{0} } { n_{f} \cos\theta_{0} + n_{0} \cos\theta_{f} } \]

The f|3 case is analogous for r12⋅, t12⋅.

Finally, we applied the Airy closed-form solution to compute the final reflected and transmitted light intensities.

The total amplitude reflection and transmission for the thin film (phase factor eiδ) are:

\[ r_{tot}^{\cdot} = \frac{ r_{01}^{\cdot} + r_{12}^{\cdot} \, e^{i \, 2\delta} } { 1 + r_{01}^{\cdot} r_{12}^{\cdot} \, e^{i \, 2\delta} } \] \[ t_{tot}^{\cdot} = \frac{ t_{01}^{\cdot} t_{12}^{\cdot} \, e^{i \, \delta} } { 1 + r_{01}^{\cdot} r_{12}^{\cdot} \, e^{i \, 2\delta} } \]

Convert amplitude to power (intensity) and apply radiometric correction for transmission:

\[ R^⋅ = | r_{tot}^⋅ |^2 \] \[ T^⋅ = ( n_3cos θ_3 / ( n_0cos θ_0 ) ) · | t_{tot}^⋅ |^2 \]

For unpolarized light, take the s/p average:

\[ R = ( R^s + R^p )/2, \quad T = ( T^s + T^p )/2 \]

After this the thin_film_RT can return the intensity ratio of the reflected light and the transmiited light compared to the incident light.

Delta BSDF Sampling

To ensure our bubble bsdf rendering returns the correct color, we construct an RGB wavelength channel and use the helper function we just implemented to obtain the reflectance and transmittance for each RGB wavelength. We then use sRGB weights to calculate the reflected intensity \(w_R\) and transmitted intensity \(w_T\) perceived by the human eye. We then calculate the light reflection probability \(p_R = w_R / (w_T + w_R)\) and use it to determine whether the sample returns reflected or transmitted light.

  • Reflection geometry: perfect mirror (local reflection). \(f_{refl} = ( R_{rgb} · p_R) / |cos θ_i|\).
  • Transmission geometry: for a single-sided thin film, approximate as straight-through (local \(ω_i = −ω_o\)).  \(f_{trans} = ( T_{rgb} · p_T ) / |cos θ_i|\).

Skybox

To enhance visual richness and avoid a monotonous background, we implemented a skybox so that bubbles could reflect and refract more varied and realistic colors instead of appearing in a blank scene. We loaded six images, each mapped to one face of the cube, forming a complete surrounding environment. In the view matrix, we set \(v(0,3) = v(1,3) = v(2,3) = 0\) to remove translation while preserving rotation, ensuring that the cube remains stationary relative to the camera and appears infinitely distant.

For path tracing, we created a PathtracingSkybox::sample method that takes a normalized direction vector as input, determines which cube face this direction intersects, converts it to 2D texture coordinates, and returns the corresponding pixel color from that face. This method enabling the path tracer to return accurate background colors when rays miss all geometry.
